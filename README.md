# Space Shooter

A classic arcade-style space shooter game built with Pygame.

## Features
- Player-controlled spaceship
- Enemy waves
- Bullet mechanics
- Collision detection
- Score tracking

## Requirements
- Python 3.7+
- Pygame

## Installation
1. Clone the repository: `git clone https://github.com/purijaniya/spaceshoot.git`
2. Install dependencies: `pip install -r requirements.txt`

## Running the Game
```bash
python main.py
```

## Controls
- Arrow Keys: Move spaceship
- Spacebar: Shoot

## Project Structure
- `main.py` - Game entry point
- `game/` - Game modules
  - `player.py` - Player spaceship class
  - `enemy.py` - Enemy spaceship class
  - `bullet.py` - Bullet class
  - `game.py` - Main game logic
